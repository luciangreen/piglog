# API reference

## `piglog/1`
Run a goal with default options.

## `piglog/2`
Run a goal with options.

## `piglog_file/3`
Load a source file, then run a goal with options.

## `piglog_convert/3`
Convert a goal into generated Piglog representation.

## `piglog_run_generated/2`
Run previously generated representation.

## `piglog_show_code/1`
Print generated code terms.

## `piglog_save_code/2`
Save generated code terms to a file.

## `piglog_clear_cache/0`
Clear generated conversion cache.

## `piglog_clear_profile/0`
Clear profiling cache.

## `piglog_unlock/1` and `piglog_lock/1`
Open or close execution gates.
